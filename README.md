# rust-flash-snippets
A comprehensive set of Rust snippets for everyone

| prefix       | description                 |
|--------------|-----------------------------|
| enum         | enum decl                   |
| fn           | fn decl                     |
| fn_async     | async fn decl               |
| fn_met       | method decl                 |
| fn_test      | test fn decl                |
| for          | for loop                    |
| fore         | for enum loop               |
| if           | if                          |
| if?!         | if-let                      |
| ife          | if-else                     |
| impl         | impl constructor/method     |
| impl_known_t | impl a known trait for type |
| let          | unmutable var declaration   |
| let_mut      | mutable var declaration     |
| loop         | infinite loop               |
| match        | match expr                  |
| match!       | match result                |
| match?       | match optional              |
| mod          | module decl                 |
| mod_test     | module test decl            |
| stru         | struct decl                 |
| trait        | trait decl                  |
| wh           | while loop                  |
| wh?          | while let loop              |
